# How to run locally
* Install the latest versions of nodejs and npm
* Clone the repository by running "git clone https://github.com/SamSchiff/spotify_group_playlist.git" (install git if you necessary)
* run "npm install" inside the project directory.  Make sure you see a file called "package.json"
* run "node server.js" and the server should start listening
* go to localhost:5000 and check out the "hello world!" that appears
